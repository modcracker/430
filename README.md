# 430.me — A Curated Cultural, Scientific & Technical Portal

This is a production-ready, highly polished, content-rich micro-encyclopedia for the premium domain name **430.me**. Rather than a static, uninspiring "this domain is for sale" screen, this page explores the multi-faceted manifestations of the number 430 across time, space, history, physics, and technology—driving organic search value and establishing brand authority, while embedding an elegant and non-intrusive domain-for-sale acquisition channel at the footer.

## Features & Craftsmanship

- **Modern Typography**: Features Space Grotesk for bold, editorial layouts, paired with Inter for clear, highly legible prose.
- **Thematic Spectral Violet Accent**: Built around a clean, near-black slate canvas (#0A0A0F) accented purely by electric violet (#7C3AED), reflecting the 430 nm wavelength peak of violet light.
- **Spectrum Visualizer**: A custom horizontal layout representing the range of visible light (380 nm to 700 nm) with high-density markers at the 430 nm photosynthetic threshold.
- **Full Static/Vercel Support**: Build-ready Next.js App Router framework with zero server-side lock-in, enabling perfect static crawls.
- **SEO Ready**: Automated Google robots.txt, fully compiled sitemap.xml, custom canvas meta configurations, OpenGraph/Twitter social cards, and structured nested JSON-LD schema markup.
- **Fluid & Accessible**: Fully responsive layout using mobile-first constraints, offering optimized touch-targets and zero layout shifts.

## Technical Requirements & Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v4 (configured completely via CSS-at-import and theme modifiers)
- **Motion**: `motion` (framing smooth typographic transitions on hero load)
- **Icons**: `lucide-react`
- **Typings**: Fully type-safe TypeScript

## Setup & Local Development

1. **Clone the project & navigate to directory**:
   ```bash
   git clone <repository_url>
   cd 430-me
   ```

2. **Install all packages**:
   ```bash
   npm install
   ```

3. **Run the local development environment**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) inside your browser to view the live responsive preview.

4. **Verify formatting & compilation**:
   ```bash
   npm run lint
   npm run build
   ```

## Vercel Deployment

Deploying this application to **Vercel** requires zero custom configurations:

1. Push your code to your GitHub / Gitlab / Bitbucket repository.
2. Visit the Vercel Dashboard at [https://vercel.com/new](https://vercel.com/new).
3. Import the repository.
4. Keep the framework preset on **Next.js** and click **Deploy**.
5. Once complete, point your brand registrar's nameservers (e.g. `430.me`) to your newly provisioned Vercel instance.
